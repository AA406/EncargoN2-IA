# Manejo de correos sospechosos y phishing

Cuando un usuario reporta un correo sospechoso, posible phishing, malware o solicitud extraña de credenciales, el caso debe tratarse con prioridad alta.

Recomendaciones iniciales:

1. No abrir enlaces ni archivos adjuntos.
2. No responder el correo.
3. No ingresar credenciales en formularios externos.
4. Tomar captura del correo o guardar encabezados si el equipo de seguridad lo solicita.
5. Reportar el caso a soporte o seguridad informática.

Criterio de decisión: los incidentes de seguridad no deben resolverse únicamente con una respuesta automática. El agente puede orientar al usuario, pero debe crear ticket y derivar a revisión humana.
