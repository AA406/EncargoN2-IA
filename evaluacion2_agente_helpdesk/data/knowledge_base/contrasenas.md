# Procedimiento de recuperación de contraseña

Cuando un usuario olvida su contraseña o no puede ingresar al sistema corporativo, debe seguir estos pasos:

1. Ingresar al portal de autoservicio de cuentas.
2. Seleccionar la opción **Restablecer contraseña**.
3. Confirmar identidad mediante correo institucional o código de verificación.
4. Crear una contraseña nueva con mínimo 10 caracteres, letras, números y símbolo.
5. Cerrar sesiones antiguas e intentar iniciar sesión nuevamente.

Si la cuenta aparece bloqueada después de varios intentos fallidos, el usuario debe esperar 15 minutos o solicitar desbloqueo a soporte.

Criterio de escalamiento: si el usuario no recibe código de verificación, si la cuenta pertenece a un cargo crítico o si existe sospecha de acceso no autorizado, se debe crear ticket y derivar a soporte humano.
