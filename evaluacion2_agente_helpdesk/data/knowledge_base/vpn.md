# Soporte para conexión VPN

La VPN permite acceder a recursos internos desde una red externa. Para resolver problemas de conexión:

1. Verificar conexión estable a internet.
2. Confirmar que el usuario esté usando la aplicación VPN oficial.
3. Validar que las credenciales institucionales estén activas.
4. Reiniciar la aplicación VPN.
5. Intentar conexión desde otra red si el error persiste.
6. Revisar si hay mantenimiento programado.

Errores comunes:

- Credenciales vencidas.
- Doble factor de autenticación no aprobado.
- Red doméstica bloqueando puertos necesarios.
- Cliente VPN desactualizado.

Criterio de escalamiento: crear ticket si la VPN impide trabajar, afecta a más de un usuario o se relaciona con cierre mensual, ventas o atención de clientes.
