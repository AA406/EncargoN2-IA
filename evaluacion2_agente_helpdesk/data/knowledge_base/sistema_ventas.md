# Incidentes en sistema de ventas y facturación

El sistema de ventas y facturación es considerado un servicio crítico porque afecta la continuidad operativa, la atención de clientes y el registro de ingresos.

Ante fallas del sistema de ventas:

1. Verificar si el problema afecta a un solo usuario o a todo el equipo.
2. Confirmar si existen mensajes de error visibles.
3. Revisar conectividad a internet y acceso al sistema.
4. Consultar si hay mantenimiento planificado.
5. Registrar hora de inicio del incidente.
6. Crear ticket con prioridad alta o crítica.

Si ningún usuario puede facturar, el agente debe crear ticket inmediatamente y derivar a soporte humano. No se debe cerrar el caso solo con una respuesta automática.
